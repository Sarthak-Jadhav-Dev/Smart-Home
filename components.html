Components Used

Arduino Uno

The brain of the system, used for processing sensor data and controlling actuators.

Rain Sensor

Detects rainfall and triggers the system to close windows automatically.

Fire Detection Module

Detects fire hazards and triggers emergency protocols like opening doors and windows.

Servo Motors

Used to automate the opening and closing of windows and doors.

Wi-Fi Module(ESP8266)

Enables remote control and monitoring via a web interface.

Smoke Sensors

Detects the presence of smoke in the room , and sends the signal for trigerring alarms and operations
